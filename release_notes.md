# Grid Up Performance Center v1.0.2 - Professional Distribution

This release marks the transition to our professional-grade distribution system, combined with critical performance and visibility updates for endurance racing.

## 🚀 Key Improvements

### 🛠️ Critical Stabilization (Definitive Fix)
- Resolved the "Black Screen" regression by optimizing the CSS and React hook lifecycle.
- Implementation of a **Safe Mode Recovery** system to ensure 100% application uptime.

### 📦 New Professional Installer
- Migration from portable builds to a formal **NSIS Setup Wizard**.
- Automatic creation of **Desktop Shortcuts** and **Start Menu** integration for a retail-ready experience.

### 📈 Telemetry Standardisation
- **Hard-locked Throttle Trace**: Throttle input is now permanently rendered in vibrant **Racing Green (#00FF88)** for professional-grade visibility.
- **Improved Performance**: Optimised telemetry bridge communication for reduced latency.

### 🎨 Dynamic Theming Engine (Final)
- Fully operational theme picker in Settings.
- Improved color propagation across all sidebars, badges, and position indicators.
- Support for **Custom Hex Codes** for team-specific branding.

## 🛡️ Security & Identity
- Hardware-locked identification via **Machine ID Integration**.
- Real-time presence tracking to prevent duplicate driver entries.
