# GRiD UP Performance Center v1.3.6

## Features & Improvements
- **Enhanced Race Control**: Now monitors incident points for the entire field. Added a 'Total Incidents' counter at the top for the active driver.
- **Global Fuel Tracking**: Consumption history is now recorded in the background, ensuring no laps are missed during page navigation.
- **Dynamic Relatives**: The Race Monitor 'Interval' card has been upgraded to a 'Relatives' view, showing live gaps ahead and behind from iRacing.
- **Track Map Highlighting**: The observed driver (or player) is now highlighted in Vibrant Yellow with a white pulse border on the track map.
- **UI Optimizations**:
  - Renamed core modules for better clarity (Session Stats, Info, Observing).
  - Increased track map size in Race Monitor.
  - Optimized Data Hub sidebar with scrollable sector analysis for long tracks like Nürburgring.
- **Branding**: Official 'G_2' icon integrated across the taskbar and application header.

## Technical Fixes
- Fixed fuel consumption history persistence issues.
- Synchronized internal versioning system.
- Improved telemetry bridge with class-based position prioritization.
